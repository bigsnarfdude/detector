# Archive

This directory contains deprecated or superseded code that is kept for historical reference.

## Contents

### `cascade/`
3-stage cascade detector architecture. This was an early design that was superseded by the simpler topic-aware causal detector approach.

**Status**: Deprecated - not actively maintained.

### `original_72_percent_experiment.py`
Early experiment that achieved 72% accuracy. Superseded by the improved experiments in `scripts/experiments/`.

**Status**: Superseded.

### `prompts/`
Archived prompt templates.

### Legacy Files
- `probe_spec.json` - Original feature specifications
- `full_probe_spec.json` - Extended specifications
- `requirements-pinned.txt` - Old pinned dependencies
- `setup_lambda.sh` - Legacy Lambda setup script

## Note

Code in this directory should not be used for new work. It is preserved for:
- Historical reference
- Reproducing old results if needed
- Understanding design evolution
